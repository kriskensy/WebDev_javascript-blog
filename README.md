# Blog Articles Manager

## Overview
Blog Articles Manager is a dynamic JavaScript-based project designed to enhance blog navigation and readability. It allows users to filter and interact with blog articles by tags and authors, providing a seamless and engaging experience for exploring content.

---

## Features
- **Dynamic Article Filtering**:
  - Filter articles by tags or authors dynamically.
- **Tag Cloud**:
  - Generates a visually appealing tag cloud with varying sizes based on tag popularity.
- **Author List**:
  - Displays a list of authors with article counts, linking to their respective articles.
- **Interactive Links**:
  - Enables clickable links for tags and authors to navigate between filtered views.

---

## Technologies Used
- **HTML5 & CSS3**: For structuring and styling the project.
- **JavaScript (ES6+)**: Core logic for interactivity and dynamic content generation.
- **Handlebars.js**: Template rendering for clean and maintainable code.
- **CSS Classes**: Dynamic class assignment for tag clouds and active link highlights.

## How It Works

- **Tag Filtering:** Clicking on a tag filters articles that contain the selected tag. Highlights the clicked tag and updates the article list dynamically.

- **Author Filtering:** Clicking on an author's name filters articles written by the selected author. Updates the list of articles while maintaining a user-friendly interface.
